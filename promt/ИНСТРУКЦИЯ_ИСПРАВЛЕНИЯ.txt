ИНСТРУКЦИЯ ПО ИСПРАВЛЕНИЮ ОШИБКИ "такой таблицы нет: books_book"

ПРОБЛЕМА:
Ошибка возникает потому, что таблицы базы данных для приложения books еще не созданы.

РЕШЕНИЕ:
1. Запустите файл fix_books_database.py двойным кликом
2. Или выполните команды в терминале:

   python manage.py makemigrations books
   python manage.py migrate
   python fix_books_database.py

АЛЬТЕРНАТИВНОЕ РЕШЕНИЕ:
Если команды не работают из-за проблем с кодировкой в PowerShell:

1. Откройте командную строку (cmd) вместо PowerShell
2. Перейдите в папку проекта
3. Выполните команды:
   python manage.py makemigrations books
   python manage.py migrate
   python fix_books_database.py

ПОСЛЕ ИСПРАВЛЕНИЯ:
- Перейдите на http://127.0.0.1:8001/books/
- Библиотека книг будет работать
- Доступны функции: каталог, чтение PDF, поиск, фильтрация

ФАЙЛЫ ДЛЯ ИСПРАВЛЕНИЯ:
- fix_books_database.py - основной скрипт исправления
- ИНСТРУКЦИЯ_ИСПРАВЛЕНИЯ.txt - эта инструкция
