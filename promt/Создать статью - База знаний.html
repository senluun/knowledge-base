Как добавить код в статью

Для добавления кода в содержание статьи используйте специальные теги:

  • <code>ваш код</code> - для простого кода
  • <code class="python">ваш код</code> - для Python
  • <code class="javascript">ваш код</code> - для JavaScript
  • <code class="html">ваш код</code> - для HTML
  • <code class="css">ваш код</code> - для CSS

Код будет автоматически обрамлен и получит кнопку копирования!
