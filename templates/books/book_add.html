Как добавить книгу

  1. Заполните обязательные поля: название, автор, категория и PDF файл
  2. Добавьте описание для лучшего понимания содержания книги
  3. Выберите теги для удобной категоризации
  4. Загрузите PDF файл - он будет доступен для чтения на сайте
  5. Нажмите "Добавить книгу" для сохранения
